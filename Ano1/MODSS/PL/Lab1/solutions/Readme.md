# Utilização do simulador

### C++
  Compilar com 
```
g++ barbershop.cpp -o barbershop 
```

  Correr com 

```
./barbershop
```

### Pyhton

Correr com 

```
python3 barbershop.py
```

### Typescript

Correr com `ts-node` (deve ser instalado na máquina `npm i -g ts-node`)

```
ts-node barbershop.ts
```

### Javascript

Correr com o `node` 

```
node barbershop.js
```
